Project Description

I Built This E-Commerce Shopping Cart Using React!

This is an e-commerce shopping cart built from scratch using React. On the site users can view a variety of products, as well as add those products to a cart. A Landing Page For A Coffee Shop Styled With CSS!

This is a simple HTML and CSS project that showcases a restaurant website called Fresno's Cafe. Project

Admin Dashboard

Project Description

This Admin Dashboard Was Styled With SCSS!

This is a dashboard for an administrative interface. The dashboard has a clean and organized layout with easy-to-access navigation links on the left-hand side. The navigation menu includes links to Home, Profile, Messages, History, Tasks, Communities, Settings, Support, and Privacy. The dashboard also features a search bar, a bell icon to receive notifications, and a profile picture with the name of the logged-in user. The right-hand side of the dashboard contains information about the user's projects, as well as options to create new projects, upload files, and share them. Additionally, there are sections for announcements and trending topics. The design of this dashboard makes it easy to manage multiple projects and stay up-to-date with important information.
