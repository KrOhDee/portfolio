Project Description

I Built This E-Commerce Shopping Cart Completely From Scratch Using React!

This is my latest project and I am really proud of it! It is an e-commerce shopping cart built from scratch using React. Project

Admin Dashboard

Project Description

This Admin Dashboard was built from scratch using HTML and Sass!

This is an admin dashboard that displays the user's current projects and much more. I used Sass for this project in order to familiarize myself with it. This project really showcases that I can lay out elements on the page, and make them look great without using a CSS framework/library. The project cards were layed out using Grid, and a lot of the other elements were layed out using Flexbox! The focus of this project is more visual than functional, as opposed to the shopping cart.
